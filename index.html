The 40/10 Workflow

How It Works

Built around a proven 50‑minute lecture format that maximises engagement and retention.

0 – 40 min

Core Instruction

The teacher delivers the main lesson. Students focus fully on learning — the platform stays in passive mode during this phase.

40 – 50 min

Self-Learning Mode

Students enter Self-Learning Mode. They submit their doubts anonymously through the secure portal. No classmate can see another's question.

Resolution

Teacher Re-Explains

The teacher reviews the private feed of questions and addresses the most frequent or critical doubts with the entire class — closing the gap in real time.

Platform Capabilities

Everything You Need

Powerful features built to remove every barrier between curiosity and understanding.

Dual-Portal Access

Separate, secure login environments for Staff and Students. Each portal is tailored with the tools and views that matter most to that role.

Core Feature

Anonymity Engine

Zero identity exposure. Students ask freely without self-consciousness.

Real-Time Feed

Questions appear on the teacher's dashboard instantly via WebSocket push — no refresh needed.

Private Teacher Feed

Only the teacher sees submitted questions. Peer visibility is completely blocked.

Topic Clustering

Smart grouping detects similar doubts and surfaces the most-asked topics to the top of the feed — so teachers can address the widest learning gaps first.
